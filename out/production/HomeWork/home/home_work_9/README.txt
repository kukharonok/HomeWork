
"Многопоточное программирование"

8. Написать новый класс с точкой входа. При запуске программы пользователь должен передать адрес папки,
 из которой мы будет считывать текстовые файлы. После запуска пользователь указывает что он хочет найти в этом тексте.
 После каждого ввода осуществляется поиск. Вводит пока не надоест. Поиск выполняется многопоточно. Один поток – один файл.
 Результат всех его поисков должен накапливаться в файла result.txt в формате «Имя файла – слово – количество»

      LogFileThread.java  - класс LogFileThread с описанием задачи для создаваемых потоков

\utils    Utils.java     - вспомогательный класс статических методов
